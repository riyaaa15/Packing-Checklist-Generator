# Trip Packing List App

A simple web application to manage packing items for different types of trips.

This project is built using HTML, CSS, and JavaScript. It allows users to select a trip type, manage items dynamically, and track packed items in real time.

---

## Features

- Select trip type (Vacation, College, Office)
- Load predefined packing items
- Add new custom items
- Delete individual items
- Mark items as packed using checkboxes
- Live counter showing packed items
- Clear current trip view
- Clean and responsive UI

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## How It Works

When a trip type is selected, the related items are loaded dynamically.  
Users can:

- Add new items
- Mark items as packed
- Delete items

The packed items counter updates automatically based on user interaction.

All items are managed using JavaScript arrays and DOM manipulation.
